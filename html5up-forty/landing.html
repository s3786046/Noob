IT Project Idea

Smart Wardrobe


Overview:

The idea of the smart wardrobe is to organize several pre-sets of clothes/outfits that are presented to you upon opening the wardrobe according to both the time of day and the current weather outside. The main purpose of the wardrobe is to save time choosing outfits and hence allow you to get straight into your day without the worry of choosing what to wear. There will of course be a manual option to create an outfit if you wish to, however it will automatically form pre-sets based on both your clothing options and personal preferences which will be entered when setting up the wardrobe.

Motivation:

The key motivating factor in coming up with this idea is my own personal struggles with choosing what to wear in the morning alongside the lack of creativity in my personal outfits. I felt that if I am having these problems that there must be without question others in this world with those same issues. The Smart Wardrobe allows for a variety of different choices in clothing that is provided to you upon opening the wardrobe, speeding up preparation and keeping your look refreshing and always suited for the occasion.

Description:

Initially, the wardrobe will custom tailored in size and capacity according to the users’ needs for the ultimate user experience. The wardrobe will be equipped with a large see through touch screen glass panel that displays and controls all the parts of the smart wardrobe, allowing for the user to set up a multitude of preferences that will allow the wardrobe to function on its own overtime. The panel will also display features such as forecasts, current fashion trends, suggest pieces of clothing that the user can purchase to complete some in fashion outfits as well as the time in the top corner of the display. The wardrobe will be linked to the internet as well as the users’ calendar, allowing for the wardrobe to be aware of the temperature and the location/dress-code of the event you will be attending. The wardrobe will have an input on the rightmost closet that accepts any clothing that the user places into it and sorts that clothing out into categories for both the wardrobe and the user to utilise. The user can setup and name his/her own pre-sets or allow the wardrobe to make its own pre-sets that the user can view and modify via the touch screen panel. The user will be given three main options: manually choosing pieces of clothing and hence using the smart wardrobe as a regular one whenever he/she wishes, allowing the smart wardrobe to come up with an outfit based on the time of the event and the occasion based off online trends and fashion databases through utilisation of present clothing that the user inputs into the system, and finally the user can set pre-sets of favourite outfits that they can bring out on command for maximum convenience. Those are the primary features of the wardrobe, however for extra costs the wardrobe can be fitted with attachments to further make the users’ experience truly unique. An example of an attachment the user could purchase is an ironing station that can be attached to the clothing input which automatically irons and handles any clothing present using robotic arms that detect and manoeuvre the clothing appropriately and carefully before placing said clothing into the wardrobe input for sorting. Although not specified above, the smart wardrobe caters for all pieces of clothing, not just torso and leg clothing. This ranges from: hats, gloves, shoes, scarves, ties etc. All of these features merge seamlessly together on the glass display to allow for the best possible user experience a wardrobe can provide on the market.

Tools and Technologies:

The technology required to make the base smart wardrobe will be very advanced. It would require a mind that is not quite AI but is incredibly malleable and responsive to the users’ command alongside having the ability to perform actions on its own when ordered to. This would require an incredible amount of coding and utilisation of a fitted computer with WI-FI access that merges together seamlessly. The material of the wardrobe will be a variety of top end woods that the user can choose from to allow for a true custom experience. The parts of the wardrobe will have to move fluidly and swiftly which again would require immense coding and high-end metal material to form hangars and racks to store the clothing. The glass display will be incredibly hard to make and will have to merge together with the other parts of the wardrobe, requiring top end coding and carpentry. The user interface of the glass display will have to be seamless and intuitive, allowing for a comfortable and incredible custom-tailored user experience that melds all the parts of the wardrobe together.

Skills required:

The skills required for the project will be top end programming to set up the glass display that will struggle greatly in the creation and integration of the interface with the main computer that controls the wardrobe. The materials for the wardrobe will have to work with each-other hence the programmers and the carpenters involved will have to work hand in hand with each-other, resulting in a great deal of communication and planning skills required.

Outcome:

The project is catered towards the high-end market. If the project is successful it will strongly impact the 1% of the world (rich people), changing the way they manage their wardrobes and speeding up their preparation times unimaginably. The project being successful would be a step in the direction of futuristic products that are aware and function automatically, opening doors for other futuristic options to appear in the market and ever so slowly upgrading and developing the technological aspect of the world.
